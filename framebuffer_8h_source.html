95  protected:
96  // In order to support unlimited lights, it may be necessary to run the lighting pass multiple times.
97  // For each run, we calculate the lighting of each light of this run. Then we read the buffers of the previous runs to get their values.
98  // Then we add the results together, and write to the diffuse and specular buffers again.
