9  /* In order to support unlimited lights, it may be necessary to run the lighting pass multiple times.
10  For each run, we calculate the lighting of each light of this run. Then we read the buffers of the previous runs to get their values.
11  Then we add the results together, and write to the diffuse and specular buffers again.
12  In order to do that, we will need 2 diffuse and specular buffers to prevent read and writing to the same buffer in the same run.
13  Otherwise, this will result in a read-write race condition and produce artifacts as multiple fragments are reading and writing to the same buffers. */
